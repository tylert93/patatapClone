Overview

This project is an imitation of a pre-existing application called Patatap. The application doesn’t serve any real purpose but is instead designed as a fun and creative outlet. The keyboard functions as a soundboard which also triggers different coloured circles to be generated. Whenever the user presses any one of the letter keys (a, b, c etc) a short sound clip is played and a circle is generated. Both the sound and the colour of the circle are specific to the key that was pressed. The circle is animated such that its size decreases and its colour changes with each frame. The position of each circle is random and independent of what key was pressed.
The application relies on two JavaScript libraries to achieve this functionality; Paper.js and Howler.js. Paper.js is an open source vector graphics scripting framework that runs on top of the HTML5 Canvas and is used to generate the circle animations. Howler.js is a library makes working with audio in JavaScript easy and reliable across all platforms. It is used to import and play the audio files when a key is pressed.
An event listener is used to detect whenever a letter key is pressed. When a letter is pressed a combination of methods are used to generate a random point within the viewport – these being ‘Point()’, ‘Point.Random()’, ‘view.size.width’ and ‘view.size.height’. The ‘Path.Circle()’ method is then used to generate a circle at this position and ‘.push()’ is used to add it to the ‘circles’ array. ‘.fillColor()’ and ‘.sound.Play()’ are used to apply a background colour to the circle and plays its associated sound respectively. Both the particular colour and sounds are fetched from the ‘keyData’ array which stores this information for each letter.
A ‘for loop’ inside of the ‘onFrame()’ method is used to change the hue and hence colour of each circle and decrease its size with every frame which passes. Once the area of a circle is less than 1 pixel, it is removed from the circles array. This prevents actions being needlessly carried out on circles which cannot be seen and is done to maximise performance.
